# SkyCash

A responsive, front-end reward dashboard built with plain HTML, CSS, and JavaScript. It is ready to publish directly with GitHub Pages.

## Run locally

Open `index.html` in any modern browser. Register a demo account to access the dashboard. All demo data is stored only in that browser via `localStorage`.

## GitHub Pages deployment

1. Create a GitHub repository and upload all project files, keeping the folder structure intact.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder, then save.
5. GitHub will provide the public URL after deployment.

## Rewarded ads

The ad boxes in `earn.html` are clearly marked placeholders. Replace their contents with approved ad-network code when you are ready. The built-in “Watch Ad” buttons are a demo only: they add coins to local storage and enforce a five-ad daily limit.

## Demo data

To reset the account, clear your browser site data/local storage for the page, then reload.
